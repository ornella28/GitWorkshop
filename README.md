# Commands used
## Task1
1. git init: initiate the ceated repository Exercise1
2. git status: check if the file created locally is showing in git##
3. git add note.txt: stage the created file 
4. git add note.txt. stage the modified text in the file note.txt
5. git commit -m "modify the hello everyone to hello all!": commit the changes made to the text so that they can be saved properly
##changed the content of the file and wrote hello class
6. git restore note.txt: restored the content of note.txt to what it was previously
7. git log: see all the changes that have been made, from the creation to the file to the modified text
8. git add . : stage all the changes
9. git commit -m "modify the hello class to hello all": commit the new changes, transfer them to git
10. git add .gitignore: staged a file .gitignore i created
11. git commit -m "added a gitignore file": transfer the gitignore file to git
12. git remote add origin https://github.com/ornella28/Exercise1.git. I linked the local repository Exercise1 to the remote repository
13. git add evaluation.txt: staged a new file created
14. git commit -m "created a new evaluation.txt file": transfer the file to git
